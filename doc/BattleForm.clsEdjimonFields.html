name The actual name of the Edjimon as prescribed by the Edjidex.
nickname The user-defined nickname. This is not used for wild edjimon.
number The Edjidex-prescribed number for the Edjimon. It is a string so that it stays at 3 digits at all times.
pics Contains the pictures for the battle screen.
rarity The number represents the odds of seeing the Edjimon in the wild. 1: 89% 2: 10% 3: 1%
speed The speed rating calculated by a formula based on level. This determines who goes first in a battle. This increases with level.
